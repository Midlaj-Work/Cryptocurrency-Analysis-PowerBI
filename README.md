# Cryptocurrency-Analysis-PowerBI
Interactive Cryptocurrency Dashboard built using Power BI

## 📌 Project Overview

This Power BI dashboard provides an interactive analysis of cryptocurrency market performance. Users can explore Market Capitalization, Trading Volume, High, Low, Average, Open, and Close prices using interactive filters and visualizations.
